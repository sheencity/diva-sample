# diva-sdk Demo

该项目为调用 diva-sdk 接口的案例

## 下载

```javascript
git clone https://github.com/sheencity/diva-sample
```

## 安装依赖

- node v14 lts
- npm v7

```javascript
npm install
```

## 运行项目

```javascript
npm start
```

项目将运行在 `http://localhost:4200/`。

## 打包项目

```javascript
npm build
```

打包文件将存储在 `dist/` 目录中，可以通过添加 `--prod` 标记打包生产环境文件。
